### Local Env Setup

0. Make sure Monoid backend server is running.

1. Copy `.env.example` with a name `.env`.
    ```
    cp .env.example .env
    ```

2. Change all values in `.env`.

3. Install dependencies using `yarn`. 
    ```
    yarn install
    ```

### Run client
```
yarn dev
```
